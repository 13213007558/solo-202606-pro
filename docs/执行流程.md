# 执行流程

## 推荐：多轮自动化

1. 复制并填写配置：

```powershell
Copy-Item config.example.json config.json
```

2. 安装依赖：

```powershell
pip install -r requirements.txt
```

3. 生成首轮提示词：

```powershell
python generate.py -n 8
```

4. 启动多轮自动化：

```powershell
node run_multi_round_auto.js --rounds 3 --batch-size 8 --wait-seconds 600
```

5. 查看结果：

- 单任务结果：`tasks/autoN/result_autoN.json`
- 汇总结果：`multi_round_results.xlsx`

项目仅保留多轮自动化流程，旧版单轮及后处理脚本已经移除。
