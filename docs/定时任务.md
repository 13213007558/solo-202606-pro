# 定时任务

使用 `schedule_run.js` 可以在指定时间启动多轮自动化。

默认命令是 3 轮、8 个窗口、每轮等待 600 秒：

```powershell
node schedule_run.js 00:00
```

指定完整命令：

```powershell
node schedule_run.js 00:00 node run_multi_round_auto.js --rounds 3 --wait-seconds 600 --batch-size 8
```

例如凌晨 2:30 执行：

```powershell
node schedule_run.js 02:30 node run_multi_round_auto.js --rounds 3 --wait-seconds 600 --batch-size 8
```

运行后会显示倒计时。若指定时间已经过去，任务会安排到第二天的同一时间。
